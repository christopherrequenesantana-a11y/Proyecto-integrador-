# Sistema de Adivinanza de Números mediante Algoritmos de Búsqueda

Integrante:
Christopher Kent Requené Santana

Carrera:
Ingeniería en Software

Objetivo del sistema:
Desarrollar un programa en Python que permita al computador adivinar un número pensado por el usuario mediante la aplicación de estructuras lógicas y repetitivas.

Descripción de funcionalidades:
- El usuario piensa un número entre 1 y 100.
- El computador propone un número.
- El usuario indica si el número es mayor, menor o correcto.
- El sistema ajusta el rango de búsqueda automáticamente.
- El proceso se repite hasta encontrar el número correcto.
- El programa muestra la cantidad de intentos realizados.
